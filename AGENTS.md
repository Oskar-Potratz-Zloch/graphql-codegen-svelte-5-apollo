# AGENTS.md

## What this repo is

A GraphQL Code Generator plugin (`src/index.ts`) that generates Svelte 5 runes-based Apollo query/mutation/subscription hooks for **Apollo Client v4**.

## Project structure

- `src/index.ts` — Plugin source (the only file you edit)
- `dist/index.js` — Compiled output (generated by `bun run compile`)
- `example/` — Test app with codegen, local schema, Svelte 5
- `example/codegen.ts` — Codegen config (TypeScript, not YAML)
- `example/schema.graphql` — Local test schema (remote endpoint is dead)
- `HANDOFF-codegen-plugin.md` — Original migration spec (completed)

## Build & verification

```sh
bun run compile          # tsc → dist/index.js
cd example && bun run types  # run codegen to validate plugin output
```

- Root `tsconfig.json` compiles `src/index.ts` → `dist/index.js` (strict, node16)
- `package.json` `main` points to `dist/index.js`
- `example/codegen.ts` references `../dist/index.js`
- `dist/` is in `.gitignore` — do not commit build output
- No linter, test suite, or CI

## Key conventions

- `src/index.ts` uses CommonJS (`module.exports`)
- The plugin returns `{ prepend: string[], content: string[] }`
- Doc constants (`*Doc`) are generated by `visitor-plugin-common`, not this plugin
- Generated output uses Svelte 5 `$state` runes (output must be `.svelte.ts`)
- All Apollo types use non-deprecated namespace-qualified forms:
  - `ApolloClient.WatchQueryOptions<TData, TVars>`
  - `ApolloClient.QueryOptions<TData, TVars>`
  - `ApolloClient.MutateOptions<TData, TVars, TCache>`
  - `ApolloClient.SubscribeOptions<TData, TVars>`
  - `ObservableQuery.Result<T>` (not deprecated `ApolloQueryResult`)
- `TypedDocumentNode` cast on all `*Doc` constants in generated wrappers
- `v.data as unknown as T` double cast in subscribe callbacks (needed for Apollo's `DataState` union)
- `gql` imported from `@apollo/client`, not `graphql-tag`

## Apollo v4 type changes (already applied)

- `WatchQueryOptions` param order: `<TData, TVariables>` (was `<TVariables, TData>`)
- `QueryOptions` same swap
- `MutationOptions` renamed to `MutateOptions`
- `SubscriptionOptions` renamed to `SubscribeOptions`
- `client.query()` infers types from `TypedDocumentNode` — no explicit `<T>` arg
- `ObservableQuery.Result<T>` replaces deprecated `ApolloQueryResult<T>`
